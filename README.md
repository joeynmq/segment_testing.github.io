# Segment Analytics.js Test Site

A lightweight web application used for testing and validating Segment Analytics.js integrations during development and troubleshooting.

## Purpose

This project serves as a local testing environment to:

- Validate Analytics.js implementation
- Test event tracking behavior
- Verify Segment event calls
- Reproduce customer integration issues
- Debug SDK behavior

## Tech Stack

- JavaScript
- HTML
- Segment Analytics.js

## Typical Test Scenarios

- Event delivery
- User identification
- Page tracking
- Integration validation
- SDK troubleshooting

## Notes

This repository is intended as a local testing environment for development and integration work rather than a production application.
